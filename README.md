# Mi Aplicación Web con Patrones de Diseño

Este proyecto es una aplicación web que utiliza dos patrones de diseño: **Singleton** y **Factory**. El objetivo es crear una aplicación funcional con un buen diseño de experiencia de usuario (UX/UI).

## Características:

- **Patrón Singleton**: Implementado para gestionar el tema (modo oscuro y claro) de la aplicación.
- **Patrón Factory**: Usado para generar tarjetas dinámicas (productos o artículos) y formularios según la selección del usuario.
- **Carrusel de Imágenes**: Un carrusel simple que permite a los usuarios navegar entre imágenes.
- **Formulario Dinámico**: El formulario cambia dinámicamente dependiendo de la selección del usuario (feedback o soporte).
